# Arcane Studio Roadmap

Arcane Studio is an umbrella suite for safer aesthetic Linux customization.

## Arcane Guard

Status: 0.1 release candidate.

Purpose:

- Inspect local PKGBUILDs
- Inspect AUR packages
- Inspect install scripts
- Inspect dotfiles/rice directories

## Arcane Themes

Status: experimental prototype.

Arcane Guard and Arcane Themes are separate modules under Arcane Studio.
Arcane Guard focuses on safety inspection.
Arcane Themes focuses on ricing/theme management.
