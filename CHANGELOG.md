# Changelog

## v0.1.0-rc1

First public release candidate.

Added:

- Arcane Guard local PKGBUILD inspection
- AUR package inspection
- shell/install script inspection
- directory scanning for dotfiles and rice repos
- risk levels, scores, and finding counts
- JSON report output
- fail-on threshold support
- basic test suite
- experimental Arcane Themes snapshot prototype

Arcane Guard is a safety-inspection tool. It does not prove code is safe. It highlights obvious red flags and review-worthy patterns.
